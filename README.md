## 📊 Project Overview
This repository contains a professional-grade **Interactive Sales Dashboard** built in Microsoft Excel. The project analyzes Adidas sales data across the United States, providing deep insights into regional performance, product profitability, and retailer efficiency.

---

## 🚀 Key Features & KPIs
Based on the analysis of the dataset, the dashboard highlights the following critical metrics:

* 💰 **Total Sales:** $120,166,650
* 📈 **Total Operating Profit:** $332,134,761
* 📦 **Total Units Sold:** 656,930
* ⚡ **Interactive Slicers:** Dynamic filtering by **Region**, **Retailer**, and **Sales Method**.

## 🔍 Detailed Analysis Breakdown

### 📍 1. Geographic Performance
* **Top Regions:** The **West** region leads in operating profit ($89.6M), followed by the **Northeast** ($68M).
* **State Rankings:** Performance is tracked across all 50 states, identifying New York and California as high-priority markets.

### 👟 2. Product Insights
Profitability is categorized across six major product lines:
* **Men's Street Footwear** (Top Performer: $82.8M Profit)
* **Women's Apparel** ($68.6M Profit)
* **Men's Athletic Footwear** ($51.8M Profit)
* **Women's Street Footwear**
* **Men's Apparel**

### 🏪 3. Retailer Channels
The dashboard tracks performance across major US retailers:
* **Foot Locker**, **Walmart**, **Sports Direct**, **Kohl's**, **Amazon**, and **West Gear**.
* **West Gear** and **Foot Locker** emerged as the highest contributors to total operating profit.

### 💳 4. Sales Methods
A comparison of distribution channel efficiency:
* **In-store:** $127.5M Profit
* **Outlet:** $107.9M Profit
* **Online:** $96.5M Profit
